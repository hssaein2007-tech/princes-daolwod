# الأمراء | al princes

بوت تيليجرام لتحميل الروابط العامة للمقاطع/الريلز/المنشورات مع اختيار الجودة.

## ما الذي يفعله؟
- واجهة `/start` فخمة مع أزرار:
  - CANAL
  - 𝐒𝐨𝐮𝐫𝐜𝐞 𝐏𝐫𝐢𝐧𝐜𝐞𝐬™
  - المطور
  - الأوامر
- عند إرسال رابط عام مدعوم، يعرض أزرار الجودة:
  - 480P
  - 720P
  - 1080P
  - أعلى جودة
- يعتمد على `yt-dlp` لذلك يحاول قراءة كثير من المواقع العامة المدعومة.

## ملاحظات مهمة جدًا
- هذا المشروع **لا** يتجاوز DRM أو القيود الخاصة بالمحتوى المدفوع/المشفر/المحمي.
- روابط البروفايل الكاملة أو التحميل الجماعي من الصفحات ليست مضمونة.
- جلب ستوريات أي مستخدم تيليجرام عشوائيًا عبر المعرف **غير متاح** في Bot API للبوتات العامة.
- إذا كان الملف أكبر من الحد، قلل الجودة أو استخدم Local Bot API لاحقًا.

## ملفات المشروع
- `bot.py` — الكود الرئيسي
- `requirements.txt`
- `Dockerfile`
- `.env.example`

## تشغيل محلي
```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
cp .env.example .env
python bot.py
```

## النشر على Railway
1. ارفع الملفات إلى GitHub.
2. افتح Railway.
3. أنشئ مشروع جديد.
4. اختر Deploy from GitHub repo.
5. اختر المستودع.
6. Railway سيبني المشروع من `Dockerfile` تلقائيًا.
7. أضف المتغيرات التالية من تبويب Variables:
   - `BOT_TOKEN`
   - `CANAL_URL`
   - `SOURCE_URL`
   - `DEVELOPER_URL`
   - `STATIC_IMAGE_URL`
8. بعد اكتمال النشر شغّل الخدمة.

## المتغيرات
- `BOT_TOKEN` توكن البوت من BotFather
- `CANAL_URL` رابط قناتك
- `SOURCE_URL` رابط السورس
- `DEVELOPER_URL` رابط المطور
- `STATIC_IMAGE_URL` رابط الصورة الثابتة لواجهة `/start`
- `MAX_FILE_SIZE_MB` الحد الأقصى للإرسال عبر البوت
- `MAX_ACTIVE_DOWNLOADS` عدد التحميلات المتزامنة

## تحسين السرعة
- اجعل `MAX_ACTIVE_DOWNLOADS=2` أو `3` فقط
- استخدم خطة Railway مناسبة
- حدّث `yt-dlp` باستمرار لأن المواقع تتغير كثيرًا
- لا ترفع الجودة دائمًا إذا كان هدفك السرعة
