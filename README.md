# Princes Telegram Bot

بوت تيليجرام جاهز للنشر على Railway لتحميل المقاطع العامة من مواقع كثيرة عبر `yt-dlp` مع واجهة عربية وأزرار جودة.

## الميزات

- واجهة `/start` فخمة مع اسم البوت والأزرار:
  - CANAL
  - 𝐒𝐨𝐮𝐫𝐜𝐞 𝐏𝐫𝐢𝐧𝐜𝐞𝐬™
  - المطور
  - الأوامر
- استقبال أي رابط فيديو عام من المواقع المدعومة في `yt-dlp`.
- قراءة الجودات المتاحة ثم عرض أزرار: 480P / 720P / 1080P / 2K حسب توفرها.
- دعم ملفات Cookies اختياري لبعض المواقع.
- دعم اختياري لرابط صفحة إنستغرام، مع زري:
  - تحميل ستوريات الشخص
  - تحميل جميع الهايلايت
- رسالة زخرفية عند عدم وجود ستوريات/هايلايت.
- Health endpoint تلقائي إذا وفّر Railway متغير `PORT`.

## قيود مهمة

- الميزة الأساسية تعمل مع **المحتوى العام**. الروابط الخاصة أو المحمية أو التي تتطلب اشتراكًا قد تحتاج Cookies أو جلسة صالحة.
- ستوريات/هايلايت إنستغرام في هذا المشروع تعتمد على `instaloader` وتحتاج **جلسة إنستغرام صالحة** عبر `IG_LOGIN_USER` و `IG_SESSIONFILE`.
- جلب ستوريات تيليجرام لأي مستخدم عبر `@username` **غير متاح في النسخة الحالية** لأن Bot API وحده لا يوفّر طريقة عامة لقراءة ستوريات المستخدمين. إذا أردتها لاحقًا تحتاج طبقة إضافية مبنية على MTProto بحساب مستخدم تملكه أنت.
- ليس كل موقع أو كل رابط مضمون 100% لأن المواقع تغيّر بنيتها باستمرار.

## النشر على Railway

1. ارفع الملفات إلى GitHub.
2. في Railway اختر **Deploy from GitHub Repo**.
3. أضف متغيرات البيئة من `.env.example`.
4. شغّل الخدمة.

## ملاحظات السرعة

- السرعة تعتمد على:
  - سرعة المصدر نفسه
  - موارد خطة Railway
  - هل الرابط يحتاج دمج فيديو + صوت عبر `ffmpeg`
- تم تفعيل خيارات مناسبة للسرعة داخل `yt-dlp` مثل `concurrent_fragment_downloads`.

## تفعيل إنستغرام

لجعل القصص والهايلايت تعمل بثبات أفضل:

1. أنشئ session file محليًا بالأمر:
   ```bash
   instaloader --login=YOUR_IG_USERNAME
   ```
2. خذ ملف الجلسة الذي يولده Instaloader.
3. ضعه في المشروع أو Volume آمن داخل Railway.
4. اضبط:
   - `IG_LOGIN_USER`
   - `IG_SESSIONFILE`

## التشغيل المحلي

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
cp .env.example .env
python bot.py
```
